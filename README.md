# MikeFlix Movie app - API

Movie API

### Description:

Creating a backend API to store information about movies and allow users to create lists of favorite movies.

### Dependencies

- Node.JS
- express
- express-validator
- passport
- passport-jwt
- passport-local
- body-parser
- mongoose
- cors
- bcrypt
- morgan
- lodash
